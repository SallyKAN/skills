# Flomo GTD Processor

Process Flomo inbox notes using GTD methodology.

## Setup

```bash
pip install -r requirements.txt
playwright install chromium
```

## Environment Variables

```bash
export FLOMO_EMAIL="your_email"
export FLOMO_PASSWORD="your_password"
export TODOIST_API_TOKEN="your_token"  # optional
```

## Usage

Invoke the skill in Claude Code to process your Flomo inbox.
